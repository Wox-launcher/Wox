# Installation

## Install

### Windows

1. Download the latest `Wox-Full-Installer.exe` from [Releases](https://github.com/Wox-launcher/Wox/releases).
2. Run the installer.
3. Wox will launch automatically after installation.

### macOS

1. Download the latest `Wox.dmg` from [Releases](https://github.com/Wox-launcher/Wox/releases).
2. Open the dmg file and drag Wox to Applications folder.
3. Open Wox from Applications folder.

### Linux

1. Download the latest `Wox-Linux.tar.gz` from [Releases](https://github.com/Wox-launcher/Wox/releases).
2. Extract the tar.gz file.
3. Run `./wox` to start Wox.

## Uninstall

### Windows

1. Open Control Panel -> Programs -> Programs and Features.
2. Find Wox and click Uninstall.
3. Remove User Data: Delete `%APPDATA%\Wox` directory.

### macOS

1. Quit Wox.
2. Remove Wox from Applications folder.
3. Remove User Data: `rm -rf ~/Library/Application\ Support/Wox`.

### Linux

1. Remove the extracted Wox directory.
2. Remove User Data: `rm -rf ~/.config/Wox`.
