# 开发环境搭建

## 推荐 IDE

- [Visual Studio Code](https://code.visualstudio.com/) - 推荐使用，因为工作区已预配置了所有必要的设置和扩展。

## 所需依赖

- 安装 [Golang SDK](https://go.dev/dl/)
- 安装 [Flutter](https://docs.flutter.dev/get-started/install)
- 安装 [Nodejs](https://nodejs.org) 和 [pnpm](https://pnpm.io/)
- 安装 [uv](https://github.com/astral-sh/uv)

## 平台特定依赖

### macOS

- 安装 [create-dmg](https://github.com/create-dmg/create-dmg)

## 快速开始

运行以下命令以设置开发环境：

```bash
make dev
```
