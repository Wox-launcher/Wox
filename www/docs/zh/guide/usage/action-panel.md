# 操作面板

操作面板为选中的结果提供额外的操作。它允许您执行除默认操作（通常是打开项目）之外的更多操作。

## 访问操作面板

1. 在主查询窗口中选择一个结果。
2. 按 `Alt + J`（或 macOS 上的 `Cmd + J`）。

## 常用操作

可用操作取决于结果的类型，但常用操作包括：

- **打开**：打开文件、应用程序或 URL。
- **复制路径**：将文件路径复制到剪贴板。
- **复制值**：将文本值复制到剪贴板。
- **在文件资源管理器中打开**：在系统文件管理器中显示文件。
- **以管理员身份运行**：以提升的权限运行应用程序（Windows）。

## 插件特定操作

插件可以定义自己的自定义操作。例如，包管理器插件可能会为包结果提供“卸载”或“更新”操作。
