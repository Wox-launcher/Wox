# 操作面板

操作面板是结果的第二步。`Enter` 执行主要动作；操作面板展示插件附加在该结果上的其他动作。

## 打开方式

1. 选中一个结果。
2. Windows/Linux 按 `Ctrl + J`，macOS 按 `Command + J`。
3. 用键盘选择动作并按 `Enter`。

## 常见动作

动作取决于插件和结果类型：

| 结果 | 常见动作 |
| --- | --- |
| 应用 | 打开、显示所在目录、复制路径、终止运行中的应用 |
| 文件 | 打开、打开所在文件夹、移入废纸篓、显示系统上下文菜单 |
| 剪贴板项 | 复制、粘贴到当前应用、收藏、编辑别名、删除 |
| 书签或 URL | 在浏览器中打开 |
| 插件商店结果 | 安装、卸载、更新、打开网站 |
| 计算器 / 转换器结果 | 复制原始结果或格式化结果 |

## 什么时候用

当默认 `Enter` 动作不是你想要的动作时，打开操作面板。它也是发现插件能力的最快方式，不需要先翻设置或文档。
