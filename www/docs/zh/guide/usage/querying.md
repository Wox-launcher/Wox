# 查询

## 基本用法

Wox 通过快捷键触发（不同平台默认快捷键不同）。打开后，您可以输入内容来搜索应用程序、文件、书签等。

## 查询结构

Wox 中的查询通常由三部分组成：

1. **触发关键字**：用于激活特定插件（例如，`wpm` 用于插件管理）。
2. **命令**：某些插件支持特定命令（例如 `wpm install` 中的 `install`）。
3. **搜索词**：您想要搜索或处理的实际内容。

示例：`wpm install wox`

- `wpm`：触发关键字
- `install`：命令
- `wox`：搜索词

## 全局触发

某些插件支持全局触发 `*`，这意味着它们可以被任何不匹配其他特定关键字的查询触发。这通常用于文件搜索或应用程序启动。

## 内置 Everything 文件搜索（Windows）

Wox 内置文件插件（触发关键字：`f`）在 Windows 上使用 Everything 引擎。使用条件：

- 安装并运行 [Everything](https://www.voidtools.com/)，让其服务处于运行并完成索引。
- 保持 Everything 在后台运行，Wox 将调用其 API 返回即时文件结果。

## 快捷键

Wox 支持各种快捷键以提高您的效率：

| 快捷键                     | 描述                               |
| -------------------------- | ---------------------------------- |
| Windows: `Alt + Space`     | 切换 Wox 显示/隐藏（默认）         |
| macOS: `Cmd + Space`       | 切换 Wox 显示/隐藏（默认）         |
| Linux: `Ctrl + Space`      | 切换 Wox 显示/隐藏（默认）         |
| `Esc`                      | 隐藏 Wox                           |
| `Up` / `Down`              | 在结果中导航                       |
| `Enter`                    | 执行选中结果的默认操作             |
| `Alt/Cmd + J`              | 打开结果的上下文菜单（操作面板）   |
| `Tab`                      | 自动补全查询                       |

## 热键

您可以在设置中自定义切换 Wox 的全局热键。

1. 打开 Wox 设置。
2. 转到 **常规** 选项卡。
3. 点击 **快捷键** 字段并按下您想要的组合键。
