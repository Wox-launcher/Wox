# 系统插件概览

系统插件是 Wox 随应用一起提供的内置工作流。从用户角度看，它们和普通插件一样：可以返回结果、提供动作、保存设置，也会参与查询路由。

## 日常插件

| 插件 | 触发方式 | 用途 |
| --- | --- | --- |
| [应用](./application.md) | 全局 | 启动应用、打开应用目录、激活运行中的应用 |
| [文件](./file.md) | `f` | 搜索已索引的文件和文件夹 |
| [网页搜索](./websearch.md) | 全局 / 搜索引擎关键字 | 使用配置好的搜索引擎搜索网页 |
| [剪贴板](./clipboard.md) | `cb` | 复用剪贴板文本和图片 |
| [计算器](./calculator.md) | 全局 / `calculator` | 计算表达式并复制结果 |
| [转换器](./converter.md) | 全局 / `calculator` | 转换单位、货币、加密货币、进制和时间 |

## 上下文插件

| 插件 | 触发方式 | 用途 |
| --- | --- | --- |
| [浏览器书签](./browser-bookmark.md) | 全局 | 打开受支持浏览器 profile 中的书签 |
| [文件夹浏览](./explorer.md) | 文件管理器或打开/保存对话框激活时全局触发 | 在文件夹之间跳转 |
| Browser | 上下文触发 | 浏览器集成可用时搜索或切换标签页 |
| Selection | 选中文本查询 | 对其他应用中的选中文本或文件执行动作 |
| MediaPlayer | `media` | 控制当前媒体播放 |

## AI 和自定义流程

| 插件 | 触发方式 | 用途 |
| --- | --- | --- |
| [AI 对话](./chat.md) | `chat` | 与配置好的模型和 Agent 对话 |
| AI Command | `ai` | 执行保存好的模型 prompt |
| [Emoji](./emoji.md) | `emoji` | 搜索并复制 Emoji |
| 插件管理器 | `wpm`, `store`, `pm` | 安装、更新、创建和查看插件 |
| 主题 | `theme` | 应用、安装、移除或生成主题 |

## 维护类插件

| 插件 | 触发方式 | 用途 |
| --- | --- | --- |
| Doctor | `doctor` | 诊断常见配置问题 |
| Update | `update`, `upgrade` | 检查或应用 Wox 更新 |
| Backup | `backup`, `restore` | 导出或恢复设置 |
| Shell | `>` / 全局命令识别 | 从 Wox 运行 shell 命令 |
| Sys | 全局 | 执行系统动作 |

## 调整插件

1. 打开 **设置**。
2. 进入 **插件**。
3. 选择要调整的插件。
4. 检查触发关键字、启用状态和插件自己的设置。

如果结果太杂，通常是多个全局插件在响应同一段文本。明确使用 `f`、`cb`、`emoji`、`wpm` 这类关键字即可收窄结果。
