# AI 对话插件

AI 对话会在 Wox 中打开模型会话。使用前先配置 AI provider。

## 快速开始

```text
chat How do I use Wox?
```

你可以继续历史对话、切换 Agent，并在选中的 Agent 支持时使用 MCP 工具。

![AI 对话插件会话](/images/system-plugin-ai-chat.png)

## 设置

| 设置 | 用途 |
| --- | --- |
| 默认模型 | 新对话使用的模型 |
| Agents | 保存 prompt、模型、图标和工具预设 |
| MCP servers | 给 Agent 使用的工具 provider |
| Auto focus | 打开聊天时聚焦输入框 |
| Fallback entry | 当其他插件没有更好结果时显示聊天入口 |

## 隐私

聊天内容会发送给选中的模型 provider。不要把密钥、私有文档或敏感选中文本发送给在线模型，除非这符合你的工作流。
