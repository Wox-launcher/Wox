# Chat 插件

Chat 让你在 Wox 内和 AI 模型对话，并支持 Agent 与 MCP 工具。

## 能做什么

- 与已配置的 AI 模型对话
- 使用 Agent（提示词/模型/工具预设）
- 在对话中调用 MCP 工具
- 保存对话历史

## 快速开始

```
chat 如何使用 Wox？
```

## 设置项

- **默认模型**：新对话默认使用的模型
- **Agents**：名称、图标、提示词、模型、工具
- **MCP 服务器**：配置可用工具
- **自动聚焦**：打开聊天时自动聚焦输入框
- **回退入口**：无匹配时显示“用 Chat 继续…”的结果

## 注意事项

- 使用前需在 **AI Provider** 中配置 API Key。
- 消息会发送给模型提供方，避免输入敏感信息。
- 在线模型需要网络连接。
