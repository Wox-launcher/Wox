# 其他系统插件

简要说明一些辅助系统插件。

## 插件列表

- **AI Command**：运行预配置的 AI 命令。输入 `ai` 查看命令并选择执行。
- **Backup**：导出或恢复 Wox 设置。
- **Browser**：搜索并切换浏览器标签页（需要浏览器集成；仅在支持的浏览器处于前台时显示）。
- **Browser Bookmark**：按名称或 URL 搜索书签。
- **Doctor**：诊断常见问题并给出建议。
- **MediaPlayer**：控制媒体播放（播放/暂停、上一首/下一首、音量）。
- **Plugin Installer**：安装与管理插件。
- **Selection**：对选中文本/文件执行操作。
- **Shell**：直接执行终端命令。
- **Sys**：系统操作（电源、设置、信息）。
- **Theme**：管理主题。
