# 计算器插件

计算器会直接在启动器里计算表达式。它监听全局输入，所以不需要关键字也可以触发。

## 快速开始

```text
100 + 200
2 ^ 10
(500 - 100) / (10 + 10)
```

按 `Enter` 使用结果。打开操作面板可以复制原始值或格式化后的值。

![计算器插件结果列表](/images/system-plugin-calculator.png)

## 显式模式

当你想查看计算历史，或其他全局插件干扰同一段查询时，使用 `calculator`：

```text
calculator 12 * 12
calculator
```

## 注意事项

- 全局触发时需要包含运算符，例如 `+`、`-`、`*`、`/` 或 `^`。
- 支持括号。
- 千分位分隔符可以在插件设置中调整。
