# WebSearch 插件

WebSearch 支持关键词搜索和默认回退搜索。

## 能做什么

- 关键词快速搜索
- 未匹配关键词时使用默认搜索
- 选中文本搜索（仅默认搜索）
- 自动获取搜索引擎图标

## 快速开始

```
Wox Launcher
→ 使用默认搜索

g Wox Launcher
→ 使用 Google 搜索
```

## 设置项

在 **WebSearch** 设置里添加搜索引擎：

- **关键词**：快捷词（如 `g`）
- **标题**：显示名称
- **URL(s)**：使用 `{query}` 占位符
- **启用**：是否显示
- **默认搜索**：未匹配关键词时使用

### URL 变量

- `{query}` 原始文本
- `{lower_query}` 小写
- `{upper_query}` 大写

### 多 URL

配置多个 URL 会依次打开多个页面。

## 注意事项

- 选中文本搜索只显示默认搜索引擎。
- 若无结果，请检查 **启用** 和 **默认搜索**。
