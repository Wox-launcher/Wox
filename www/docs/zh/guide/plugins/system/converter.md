# Converter 插件

Converter 用于单位、货币、加密货币和进制转换，也支持简单表达式计算。

## 支持范围

- **单位**：长度、重量、温度、时间
- **进制**：bin、oct、dec、hex
- **法币**：USD、EUR、GBP、JPY、CNY、AUD、CAD
- **加密货币**：BTC、ETH、USDT、BNB
- **运算**：`+ - * /` 可混合单位计算

## 快速开始

```
1km to m
100lb to kg
10 to bin
1 btc to usd
100 usd + 50 usd
```

## 使用提示

- 使用 `to` 或 `in` 指定目标单位。
- 进制转换仅支持整数，且需要明确目标（如 `255 to hex`）。
- 只输入货币/加密货币时，会换算为系统默认货币。

## 汇率与价格

- 法币汇率约每小时更新，加密货币价格约每分钟更新。
- 启动时使用近似值，后台自动刷新。
- 离线时使用上次缓存结果。
