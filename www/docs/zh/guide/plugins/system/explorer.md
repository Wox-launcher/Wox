# Explorer 插件

Explorer 用于在资源管理器/Finder 中快速导航，也能在打开/保存对话框中跳转文件夹。

## 能做什么

- 文件管理器激活时，搜索当前目录下的条目
- 打开/保存对话框激活时，推荐常用文件夹
- 支持 Windows 和 macOS

## 快速开始

**文件管理器中**

```
Documents
Downloads
project
```

选择结果即可导航。`Ctrl+Enter` 在新窗口打开所在文件夹。

**打开/保存对话框中**

对话框激活时打开 Wox，会看到推荐路径：
- 该应用的最近使用文件夹
- 当前打开的 Finder 窗口
- 常用系统目录（桌面、文档、下载等）

## 注意事项

- 仅在资源管理器/Finder 或打开/保存对话框激活时显示结果。
- 没有可配置项。
