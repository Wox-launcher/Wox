# Emoji 插件

Emoji 插件提供快速的 Emoji 表情符号搜索和插入功能。

## 功能特点

- **海量 Emoji**: 包含数千个 Emoji 表情
- **多语言支持**: 支持多种语言的 Emoji 名称和分类
- **网格布局**: 以网格形式展示，方便浏览
- **AI 匹配**: 可选的 AI 辅助搜索，支持自然语言描述
- **使用统计**: 记录 Emoji 使用频率，优先显示常用的
- **快速复制**: 一键复制到剪贴板

## 基本使用

### 搜索 Emoji

1. 打开 Wox
2. 直接输入 Emoji 名称或关键词
3. 浏览搜索结果（网格布局）

```
smile
→ 显示所有与 "smile" 相关的 Emoji

cat
→ 显示所有与 "cat" 相关的 Emoji

heart
→ 显示所有与 "heart" 相关的 Emoji
```

### 使用触发关键词

也可以使用触发关键词 `emoji`：

```
emoji smile
→ 显示与 "smile" 相关的 Emoji
```

## 显示格式

### 网格布局

Emoji 插件使用网格布局展示搜索结果：
- 每行显示 12 个 Emoji
- 每个 Emoji 显示为图标，不显示标题
- 方便快速浏览和选择

### 默认行为

- 不显示 Emoji 标题
- 只显示 Emoji 图标
- 鼠标悬停可查看详细信息（如果支持）

## AI 匹配功能

### 启用 AI 匹配

1. 打开 Wox 设置
2. 找到 **Emoji** 插件
3. 勾选 **启用 AI 匹配**
4. 选择 AI 模型（如果需要）

### AI 匹配的优势

启用 AI 匹配后，可以使用自然语言描述来查找 Emoji：

```
一个开心的笑脸
→ 😄

红色的爱心
→ ❤️

哭泣的脸
→ 😢

绿色的勾
→ ✅

太阳和云
→ ⛅
```

### AI 模型选择

可以在插件设置中选择不同的 AI 模型：
- 不同模型可能有不同的匹配效果
- 建议使用默认模型
- 如果匹配效果不理想，可以尝试其他模型

## 搜索技巧

### 关键词搜索

使用 Emoji 名称的一部分进行搜索：

```
face
→ 显示所有面部表情

love
→ 显示所有与爱相关的 Emoji

color
→ 显示所有彩色的 Emoji
```

### 模糊匹配

即使不完全准确也能找到相关的 Emoji：

```
smil
→ 可能匹配: 😄 😊 😃

heart
→ 可能匹配: ❤️ 💕 💖
```

### 组合搜索

组合多个关键词来缩小范围：

```
face happy
→ 快乐的面部表情

animal cat
→ 动物类中的猫
```

## 使用统计

### 使用频率

Emoji 插件会记录 Emoji 的使用频率：
- 使用次数越多，优先级越高
- 常用的 Emoji 会排在前面
- 新使用的 Emoji 会快速提升优先级

### 查看统计

使用频率在后台记录，不直接显示。搜索时会自动优先显示常用的 Emoji。

## 配置选项

### AI 匹配设置

在插件设置中可以配置：

- **启用 AI 匹配**: 是否使用 AI 辅助搜索
- **AI 模型**: 选择使用的 AI 模型

**建议**:
- 日常使用：可以禁用 AI 匹配以提升速度
- 需要自然语言搜索：启用 AI 匹配

## 常见问题

### 为什么搜索不到某个 Emoji？

1. **检查拼写**: 确认 Emoji 名称拼写正确
2. **使用关键词**: 尝试使用更通用的关键词
3. **启用 AI 匹配**: 如果知道描述但不知道名称，启用 AI 匹配
4. **检查语言**: 某些 Emoji 可能在其他语言中有不同的名称

### AI 匹配不准确怎么办？

1. **尝试不同的描述**: 使用更具体或不同的描述词
2. **更换 AI 模型**: 尝试使用不同的 AI 模型
3. **禁用 AI 匹配**: 使用传统关键词搜索

### Emoji 不显示？

1. **检查系统支持**: 确认操作系统支持该 Emoji
2. **更新系统**: 某些新 Emoji 可能需要更新的系统版本
3. **检查字体**: 确认系统字体支持该 Emoji

### 如何快速插入多个 Emoji？

1. 搜索并复制第一个 Emoji
2. 粘贴到目标位置
3. 搜索并复制第二个 Emoji
4. 继续粘贴

## 技术说明

Emoji 插件使用内置的 Emoji 数据库：
- 数据嵌入在插件中

- 支持多种语言
- 网格布局由插件特性控制

AI 匹配功能：
- 使用 AI 模型处理自然语言描述
- 返回匹配的 Emoji 列表
- 可能需要网络连接（取决于 AI 模型）
