# 使用 AI 创建主题

> 请注意，此功能需要配置 AI 设置才能使用。如果您尚未配置 AI 设置，请参阅 [AI 设置](./settings.md) 文档。

Wox 提供了一项独特的功能，允许您使用 AI 创建主题。可以使用 `theme ai` 命令后跟您的主题描述来访问此功能。AI 将解释您的描述并据此创建主题。

## 用法

例如，如果您想创建一个马卡龙配色的主题，可以使用命令：

```
theme ai macaroon color
```

AI 随后将生成一个匹配马卡龙配色的主题。

此功能提供了一种快速简便的方法来为您的 Wox 启动器创建独特且个性化的主题。享受 AI 驱动的主题创作带来的灵活性和创造力。
