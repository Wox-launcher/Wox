# 安装

## 安装

### Windows

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 `Wox-Full-Installer.exe`。
2. 运行安装程序。
3. 安装完成后 Wox 会自动启动。

### macOS

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 `Wox.dmg`。
2. 打开 dmg 文件并将 Wox 拖入应用程序文件夹。
3. 从应用程序文件夹打开 Wox。

### Linux

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 `Wox-Linux.tar.gz`。
2. 解压 tar.gz 文件。
3. 运行 `./wox` 启动 Wox。

## 卸载

### Windows

1. 打开控制面板 -> 程序 -> 程序和功能。
2. 找到 Wox 并点击卸载。
3. 删除用户数据：删除 `%APPDATA%\Wox` 目录。

### macOS

1. 退出 Wox。
2. 从应用程序文件夹中移除 Wox。
3. 删除用户数据：`rm -rf ~/Library/Application\ Support/Wox`。

### Linux

1. 删除解压的 Wox 目录。
2. 删除用户数据：`rm -rf ~/.config/Wox`。
