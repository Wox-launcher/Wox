# 安装

## 安装

### Windows

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 Windows 安装包。
2. 运行安装程序并按照提示完成安装。
3. 安装完成后 Wox 会自动启动。

### macOS

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 macOS 版本。
2. 打开下载的镜像并将 Wox 拖入「应用程序」文件夹。
3. 从「应用程序」文件夹打开 Wox。

### Linux

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 Linux 压缩包。
2. 解压压缩包。
3. 运行 `./wox` 启动 Wox。

## 卸载

### Windows

1. 打开控制面板 -> 程序 -> 程序和功能。
2. 找到 Wox 并点击卸载。
3. 删除用户数据：如需清理配置，请删除 `%USERPROFILE%\\.wox` 目录。

### macOS

1. 退出 Wox。
2. 从应用程序文件夹中移除 Wox。
3. 删除用户数据：如需清理配置，请执行 `rm -rf ~/.wox`。

### Linux

1. 删除解压的 Wox 目录。
2. 删除用户数据：如需清理配置，请执行 `rm -rf ~/.wox`。
