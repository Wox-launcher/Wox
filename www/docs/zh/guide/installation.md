# 安装

Wox 是一个自包含的应用程序，无需安装，下载后即可直接运行。

## 下载与运行

### Windows

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 Windows 版本。
2. 解压到任意目录。
3. 双击 `Wox.exe` 即可运行。

### macOS

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 macOS 版本。
2. 打开下载的镜像并将 Wox 拖入「应用程序」文件夹。
3. 从「应用程序」文件夹打开 Wox。

### Linux

1. 从 [Releases](https://github.com/Wox-launcher/Wox/releases) 下载最新的 Linux 版本。
2. 解压到任意目录。
3. 运行 `./wox` 启动 Wox。

## 卸载

Wox 是自包含的应用，卸载只需删除程序文件即可。

### Windows

1. 删除 Wox 所在的文件夹。
2. 删除用户数据：如需清理配置，请删除 `%USERPROFILE%\.wox` 目录。

### macOS

1. 退出 Wox。
2. 从应用程序文件夹中移除 Wox。
3. 删除用户数据：如需清理配置，请执行 `rm -rf ~/.wox`。

### Linux

1. 删除 Wox 所在的目录。
2. 删除用户数据：如需清理配置，请执行 `rm -rf ~/.wox`。
