# 你知道吗：Wox 可以找回类似 Launchpad 的应用网格

macOS Tahoe 改变了原来 Launchpad 的使用习惯。如果你还是更喜欢那种一屏应用图标的熟悉感觉，Wox 可以通过内置应用插件提供一个类似的入口。

唤起 Wox 后输入：

```text
app launchpad
```

应用插件会在这个命令下切换到网格布局，把已安装应用显示成较大的图标，并在图标下方显示应用名称。这个效果更接近经典 macOS Launchpad：先扫一眼图标网格，认出应用，再打开它。

![Wox 应用启动台网格](/images/did-you-know-wox-app-launchpad.png)

当你记得应用图标、但不想输入应用名称时，这个入口会很顺手。刚装好系统或新机器时，它也适合用来重新熟悉当前安装了哪些工具。

普通的 Wox 应用搜索仍然适合快速输入和启动。`app launchpad` 是单独的命令：默认应用查询保持紧凑、偏搜索，而启动台模式提供一个更适合浏览的视觉网格。

如果想更快打开它，可以给 `app launchpad` 绑定一个快捷键查询。这样一个快捷键就能直接打开应用网格，让 macOS Tahoe 用户在 Wox 里找回熟悉的 Launchpad 感觉。
