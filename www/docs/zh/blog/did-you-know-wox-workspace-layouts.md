# 你知道吗：Wox 可以一键恢复工作区布局

有些工作每次开始都要摆同一组窗口：编辑器放一边，终端放下面，浏览器或笔记放到另一块屏幕。Wox 现在可以把这套排列保存成工作区布局，再从启动器里一键恢复。

![Wox 工作区布局编辑器](/images/did-you-know-wox-workspace-layouts.png)

先在 Window Manager 插件设置里创建一个工作区：

1. 打开 Wox 设置。
2. 进入插件，然后打开 Window Manager。
3. 在工作区布局里新增一个工作区。
4. 选择每块屏幕，挑一个窗口布局，再把应用绑定到对应位置。

保存后，唤起 Wox 输入：

```text
window group
```

选择刚刚保存的工作区。Wox 会把匹配到的应用窗口移动到配置好的位置。如果某个应用还没打开，并且 Wox 知道它的应用路径，也可以先打开应用，再等窗口出现后放到对应位置。

这个功能适合重复出现的上下文：写代码、写文档、开会、直播，或者任何你经常手动重新摆放的窗口组合。

如果想做到一个快捷键直接恢复，可以给精确的工作区查询绑定一个快捷键查询，例如：

```text
window group Work
```

当这个查询只匹配到一个工作区时，可以配合静默执行，直接应用布局而不显示启动器。

工作区布局会基于当前显示器配置排列普通应用窗口。如果应用阻止系统窗口管理，或者保存时的某块屏幕没有连接，Wox 会尽量应用可处理的部分，并提示哪些窗口没有移动成功。
